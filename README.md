BowlingMechanic
=================

![readme image][1]
![readme image][2]
![readme image][3]

####Gameplay
Single player. Toss infinite bowling balls at pins to knock over. They howl sounds from upon impact. Sun changes sky colours. 

####Setup
Plays in web browser.
